## üöÄ How to Contribute

### 1. Fork the Repository

Click the **"Fork"** button at the top-right of the [GitHub repository](https://github.com/DomainTyler/Quick-Tools-Launcher-by-DomainTyler) to create your own copy.

### 2. Clone Your Fork

Open a terminal or command prompt and run:

```bash
git clone https://github.com/YOUR_USERNAME/Quick-Tools-Launcher-by-DomainTyler.git
cd Quick-Tools-Launcher-by-DomainTyler
```

### 3. Create a New Branch

It's best to make your changes in a separate branch:

```bash
git checkout -b feature/your-feature-name
```

### 4. Make Your Changes

- Keep your code clean and well-commented
- Follow the existing coding style
- Make sure the app still runs without errors
- If adding a new feature, consider adding a brief explanation or usage tip in the `README.md` or in the UI

### 5. Commit and Push

```bash
git add .
git commit -m "Add: Describe your changes here"
git push origin feature/your-feature-name
```

### 6. Submit a Pull Request

Go to your forked repository on GitHub, click **"Compare & pull request"**, and write a clear, helpful description of what you did.
